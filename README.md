# At3aby - Legal Services Marketplace

> A platform connecting clients with specialized lawyers through case management and bidding system.

## About

This repository contains selected code samples from the At3aby platform, developed during my work at TopBusiness. The project involved building a comprehensive legal services marketplace connecting clients with lawyers.

## Project Overview

The platform enables:
- Clients to create legal cases with specific case types
- Intelligent routing to specialized lawyers based on case type
- Real-time chat between clients and lawyers
- Lawyers to submit offers on cases
- Case assignment and progress tracking
- Emergency case system for urgent matters

## Key Features

- **Lawyer Directory:** Search, ratings, and reviews
- **Case Management:** Current, completed, and emergency cases
- **Bidding System:** Multiple lawyers can offer on each case
- **Wallet System:** Built-in wallets for clients and lawyers
- **Monetization:** Subscription packages, promo codes with points conversion
- **Community:** Posts, comments, and engagement features
- **Marketplace:** Digital legal document templates

## Technologies Used

- **Framework:** Laravel
- **Database:** MySQL
- **Real-time:** Chat system
- **Payment:** Multi-wallet integration
